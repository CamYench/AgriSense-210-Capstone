Why is this important?

Inefficient Irrigation

70% of freshwater is used for agriculture, 50% of which
is wasted in inefficient irrigation.[1][2]

Wasted Fertilizer

Over 40% of fertilizer is wasted in unabsorbed runoff.[3]

Loss of Farmable Land

1/3 of the world's farmable land has disappeared in
the last 4 decades. Major contributing factors are
overfertilization and unsustainable agriculture practices.[4]

Greenhouse Gas Emissions

Agriculture accounts for 22% of global greenhouse gas emissions.[5]

Precision Agriculture

Farmers using precision agriculture technologies have achieved:[6]

  • 4% reduction in water use
  • 7% increase in fertilizer placement efficiency
  • 9% reduction in herbicide and pesticide use
  • 6% reduction in fossil fuel use
  • 4% increase in crop production

Our Methodology

Data Acquisition: Download Landsat and Cropscape data via APIs from nassgeodata.gmu.edu (Cropscape) and espa.cr.usgs.gov (Landsat) using custom ETL scripts; jobs are scheduled in Sagemaker and run automatically

Reprojection and Resampling: Reproject and resample the data to ensure consistent spatial resolution and coordinate systems. Custom Python functions handle these transformations, preparing the data for uniform analysis.

Masking: Apply a mask to the reprojected and resampled data to isolate specific areas of interest. A threshold creates the mask, which filters the target raster data.

Loading to S3: Upload the processed and masked data to Amazon S3.

CNN Feature Extractor: Extracts features from each landsat image to identify regions of high/low productivity 4 x Conv2d layers with 2x2 max pooling

LSTM: Takes output from CNN feature extractor passed through single layer, 64 hidden size LSTM Concatenated with additional time features before passed to output layer (cyclical encoding for month/day-of-year, cumulative volume)

Compute: Amazon Sagemaker is a fully managed service that provides developers and data scientists with the ability to build, train, and deploy machine learning models quickly.

Storage / Database: Amazon S3 is a scalable object storage service that provides secure, durable, and highly available storage for our wide range of data types.

Front End: Streamlit: Streamlit is an open-source Python library that simplifies the creation of interactive, web-based dashboards. SageMaker allows for hosting Streamlit servers + connections to our models for real-time predictions and interactive data analysis.
